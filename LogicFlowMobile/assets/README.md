# Assets

Coloca aquí los siguientes archivos antes de compilar:

- `icon.png` — 1024×1024 px (ícono de la app)
- `splash.png` — 1284×2778 px (pantalla de carga)
- `adaptive-icon.png` — 1024×1024 px (ícono adaptativo Android)
- `favicon.png` — 48×48 px (favicon web)

Puedes generar versiones temporales con cualquier imagen cuadrada usando:
`npx expo-asset generate`
