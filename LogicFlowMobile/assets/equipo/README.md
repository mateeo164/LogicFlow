# Fotos del equipo (móvil)

Reemplaza los placeholders por la foto formal de cada integrante, manteniendo **exactamente** estos nombres (formato `.png`):

| Archivo                     | Integrante          |
|-----------------------------|---------------------|
| `mateo-almeida.png`         | Mateo Almeida       |
| `jonathan-cunguan.png`      | Jonathan Cunguan    |
| `kevin-guaman.png`          | Kevin Guamán        |
| `washington-pizarra.png`    | Washington Pizarra  |

Notas:
- Deben ser `.png` con estos nombres (Expo/Metro los referencia con `require()` en `ProfileScreen.tsx`).
- Cuadradas, p. ej. 400×400 px. Se muestran en círculo automáticamente.
- Los archivos actuales son placeholders (el icono de la app); simplemente sobrescríbelos.
- Tras reemplazarlas, reinicia el bundler de Expo para que tome las imágenes nuevas.
